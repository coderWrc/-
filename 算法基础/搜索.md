# [枚举子集（递归实现指数型枚举）](https://www.luogu.com.cn/problem/B3622)  
[top](#枚举子集递归实现指数型枚举)
```
#include <iostream>

using namespace std;

int n;
string path;

void dfs(int pos)
{
	if (pos > n)
	{
		cout << path << endl;
		return;
	}
	path += 'N';
	dfs(pos + 1);
	path.pop_back();
	path += 'Y';
	dfs(pos + 1);
	path.pop_back();
}

int main()
{
	cin >> n;
	dfs(1);

	return 0;
}
```
